## Surprise
Surprise is a tool to manage social meida giveaways. It's built on top of the NEAR protocol.

## Start the project

Backend:

```
> cd ./server
> yarn start
```

Frontend

```
> yarn start:testnet
```

Open the browser with: `127.0.0.1:3000`. Please use `127.0.0.1` instead of `localhost`.
